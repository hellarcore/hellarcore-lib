## .serialize(unsafe)

**Description**: Retrieve a hex string that can be used with hellard's CLI interface (decoderawtransaction, sendrawtransaction)
Flags that can be used to skip tests:

- `disableSmallFees`: disable checking for fees that are too small

## .from(utxo, pubkeys, threshold)

**Description**: Add an input to this transaction.  
This is a high level interface to add an input, for more control, use Transaction#addInput.

Can receive, as output information, the output of hellard's `listunspent` command, and a slightly fancier format recognized by hellarcore:

```json
{
  "address": "yYo3PeSBv2rMnJeyLUCCzx4Y8VhPppZKkC",
  "txId": "a477af6b2667c29670467e4e0728b685ee07b240235771862318e29ddbe58458",
  "outputIndex": 0,
  "script": null,
  "hellars": 1020000
}
```

Where `address` can be either a string or a hellarcore Address object. The same is true for `script`, which can be a string or a hellarcore Script.

**Parameters**:

| parameter     | type         | required | Description                                            |
| ------------- | ------------ | -------- | ------------------------------------------------------ |
| **utxo**      | Array/Object | yes      | Array of utxos or single utxo object or representation |
| **pubkeys**   | Array        | no       | In case of a multisigUtxo.                             |
| **threshold** | Number       | no       |                                                        |

**Returns**: Transaction
